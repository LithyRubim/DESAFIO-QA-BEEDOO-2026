# DESAFIO-QA-BEEDOO-2026

* Objetivo da Aplicação: Cadastro e Listagem de Cursos disponíveis na página.
* Fluxos Disponíveis: Cadastro e Listagem de Cursos.
* Pontos Críticos:
  1. Cadastro: Campos obrigatórios, confirmação de cadastro, mensagens de erro por dados inválidos ou campos obrigatórios não preenchidos.
  2. Listagem: Informações exibidas corretamente, busca por cursos cadastrados, atualização da lista após cadastro.
